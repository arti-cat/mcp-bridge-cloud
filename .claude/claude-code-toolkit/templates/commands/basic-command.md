# Basic Slash Command Template

This is a minimal slash command without frontmatter.

## Usage
Simply write your prompt instructions here. The command will be available as `/filename` where filename is the name of this markdown file (without .md extension).

## Template Structure
```
Your instructions to Claude go here.

You can use:
- Markdown formatting
- Multiple paragraphs
- Code blocks
- Lists

Claude will follow these instructions when the command is invoked.
```

## Example
If this file is named `review-code.md`, it becomes `/review-code`
